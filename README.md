# 🔥 라이브 하이라이트 메이커 (무설치판)

**긴 라이브 방송 다시보기를 자동으로 분석해 명장면(하이라이트)만 뽑아내고,**
**자막·화면전환·배경음악까지 얹어 업로드용 완성 영상으로 만들어 주는 한국어 영상 제작 도구입니다.**

Whisper 음성 인식과 ffmpeg 기반 · 파이썬을 몰라도 쓸 수 있는 **그래픽 UI(GUI)** 제공

</div>

---

## 🎯 이 프로그램은 무엇인가요?

몇 시간짜리 라이브 방송을 통째로 다시 보며 편집하는 건 지치는 일입니다. 이 도구는 그 과정을 자동화합니다.

> **방송 다시보기 URL 하나만 넣으면 → 오디오가 뜨거웠던 순간(하이라이트)을 스스로 찾아 → 목표 길이에 맞춰 짧은 요약 영상으로 만들고 → 자막까지 달아 줍니다.**

게임 방송 하이라이트 편집을 염두에 두고 만들었지만, 어떤 라이브/영상에도 그대로 쓸 수 있습니다.

## ✨ 주요 기능

GUI(`요약기_gui.py`)는 네 개의 탭으로 구성됩니다.

### 1️⃣ 영상 요약 탭 — 방송을 하이라이트로

- 방송 다시보기 **URL을 넣거나**, 직접 **녹화한 로컬 영상 파일을 불러오면** **오디오 에너지가 높은 하이라이트 구간**을 자동으로 찾습니다. (URL은 다운로드부터, 로컬 파일은 다운로드를 건너뛰고 바로 분석)
- 목표 길이(예: 10분)에 맞춰 하이라이트를 골라 **요약 영상**을 만듭니다.
- Whisper로 **자막(SRT)** 을 자동 생성하고, **유튜브 챕터 텍스트**(`_chapters.txt`)도 함께 만들어 설명란에 바로 붙여넣을 수 있습니다.
- **「구간 후보만 분석」** 을 켜면 영상을 만들지 않고 하이라이트 후보 구간만 빠르게 뽑아 **수동 하이라이트 탭에 자동으로 채워줍니다** → 구간을 직접 확인·수정한 뒤 만들 수 있습니다.
- 하이라이트 사이에 넣을 **화면 전환**(없음·암전·화이트 플래시)과 **전환 효과음**(없음·휙·스와이프·삑·팝·임팩트)을 골라 넣을 수 있습니다.
- **채널 마크(로고 이미지)** 를 골라 원하는 모서리(좌상·우상·좌하·우하)에 얹을 수 있습니다. 마크는 **본영상에만** 새겨지고, 완성 탭에서 붙이는 인트로/아웃트로엔 나오지 않습니다.
- **원본 영상 보관 폴더**를 지정하면 내려받은 원본 영상을 삭제하지 않고 보관합니다. → 나중에 **수동 하이라이트 탭**에서 재편집할 수 있습니다.
- 결과물: `제목_summary.mp4`(요약 영상), `제목_summary.srt`(자막), `제목_chapters.txt`(유튜브 챕터)

### 2️⃣ 수동 하이라이트 탭 — 내 영상 + 내가 고른 구간

- 이미 가지고 있는(로컬) 영상 파일과 **직접 입력한 하이라이트 시간대**만으로 요약 영상을 만듭니다. 다운로드·오디오 분석 단계를 건너뜁니다.
- 시간대는 한 줄에 하나씩 `시작 - 끝` 형식으로 입력합니다. `1:23 - 2:05`, `83 - 125`, `00:01:23,000 --> 00:02:05,000`(SRT 스타일)처럼 `SS` / `MM:SS` / `HH:MM:SS`를 모두 지원합니다.
- 각 줄 뒤에 `| 소제목`을 붙이면(예: `1:23 - 2:05 | 다운그레이드`) 그 하이라이트 구간 동안 **마크 아래에 소제목**이 표시됩니다.
- 요약 탭과 똑같이 **화면 전환·효과음·채널 마크**를 넣고, 원하면 **자막(SRT)** 도 자동 생성합니다.
- 결과물: `이름_highlight.mp4`, `이름_highlight.srt`(자막 켤 때), `이름_chapters.txt`(유튜브 챕터)

### 3️⃣ 쇼츠 만들기 탭 — 구간을 골라 9:16 세로 영상으로

- 영상에서 원하는 구간을 골라 **1080x1920 세로 쇼츠 영상**을 만듭니다. (유튜브 Shorts / Instagram Reels / TikTok 규격)
- 세로 변환 방식 2가지: **중앙 크롭**(화면 가운데 확대) / **블러 배경**(원본 화면 그대로 + 위아래 블러 채움)
- **자막 자동 생성**을 켜면 Whisper 자막을 쇼츠 스타일의 **크고 굵은 캡션**으로 새겨넣습니다.
- 여러 구간을 입력하면 하드컷으로 이어붙입니다. (총 3분 이하 권장)
- 결과물: `이름_shorts.mp4`(세로 영상), `이름_shorts.srt`(자막 켤 때)

### 4️⃣ 완성 영상 만들기 탭 — 업로드용으로 마무리

- **요약 영상 + 자막(SRT) + 썸네일**을 골라 하나의 완성 mp4로 합칩니다.
- 자막을 화면에 **새겨넣기(하드섭)** → 어디서 재생해도 자막이 보입니다.
- 썸네일을 **① 인트로 클립**으로 앞에 붙이고, **② mp4 표지(커버 아트)** 로도 삽입합니다.
- **인트로 영상 / 아웃트로 영상**(별도 mp4)을 본편 앞·뒤에 붙일 수 있습니다. 해상도·비율이 달라도 본편 규격에 맞춰 자동 변환해 이어붙입니다.
- **배경음악(BGM)** 을 골라 넣으면 완성 영상 **전체 길이에 맞춰 자동으로 반복(짧으면)·컷(길면)** 되어 깔립니다. 볼륨을 낮춰(기본 0.25) 원래 말소리를 덮지 않게 섞습니다.
- 인트로 길이·자막 크기·배경음악 볼륨·각 옵션 on/off 조절 가능.

## 💡 추천 워크플로우

AI 자막은 완벽하지 않습니다. 아래 순서로 진행하면 **자막까지 정교하게 다듬은 완성 영상**을 만들 수 있습니다.

<div align="center">
<img src="assets/workflow.svg" alt="추천 워크플로우: STEP 1 요약 → STEP 2 자막 수정 → STEP 3 완성" width="100%">
</div>

```
① 영상 요약 탭  →  요약 영상(.mp4) + 자막 파일(.srt) 자동 생성
        ↓
② 메모장으로 .srt 파일을 열어 틀린 단어·표현 직접 수정
        ↓
③ 완성 영상 만들기 탭  →  수정된 .srt + 썸네일 + 요약 영상을 합쳐 완성!
```

> **Tip:** SRT 파일은 메모장(또는 Notepad++)으로 열면 됩니다. 자막 텍스트만 고치고 저장하면 끝입니다. 타임코드(숫자)는 건드리지 않아도 됩니다.

---

## 🚀 3분 빠른 시작

1. **`실행.bat` 더블클릭** → 프로그램 창이 열립니다.
2. **`영상 요약` 탭** → 방송 다시보기 URL 붙여넣기 → **「다운로드 & 요약」** 클릭 → 요약 영상(.mp4)과 자막(.srt) 자동 생성.
3. *(선택)* 메모장으로 **`.srt`** 열어 틀린 단어만 고치고 저장.
4. **`완성 영상 만들기` 탭** → 방금 만든 영상 + 자막 + 썸네일 선택 → **「완성 영상 만들기」** 클릭.
5. 업로드용 **완성 mp4** 가 저장됩니다. 끝! 🎉

### STEP 1 — 영상 요약 탭

아래는 실제 프로그램 화면입니다. 번호를 따라가면 됩니다.

<div align="center">
<img src="assets/ui-summarize.svg" alt="영상 요약 탭 화면 안내 — 각 버튼과 입력칸의 역할" width="100%">
</div>

| 항목 | 설명 | 기본값 |
|------|------|--------|
| URL 또는 로컬 파일 | 요약할 다시보기 주소 붙여넣기, 또는 **「파일 선택」**으로 직접 녹화한 영상 파일 불러오기 | — |
| 출력 폴더 | 결과 파일이 저장될 폴더 | `output` |
| 목표 길이(분) | 만들고 싶은 요약본의 길이 | `10` 분 |
| Whisper 모델 | 자막 정확도 (클수록 정확·느림) | `small` 권장 |
| 언어 | 음성 언어 | `ko` (한국어) |
| 화질 | 360 / 480 / **720** / 1080 | `720` 권장 |
| 피크 앞 확장(초) | 하이라이트 시작점 앞에 더 붙이는 시간 | `5` 초 |
| 피크 뒤 확장(초) | 하이라이트 끝점 뒤에 더 붙이는 시간 | `20` 초 |
| 같은 장면 묶기(초) | 이 값보다 가까운 구간은 자연스럽게 이어붙임 | `8` 초 |
| 화면 전환 | 하이라이트 사이 전환 방식: 없음 / 암전 / 화이트 플래시 | `암전` |
| 전환 효과음 | 전환 지점 효과음: 없음 / 휙 / 스와이프 / 삑 / 팝 / 임팩트 | `휙` |
| 채널 마크 이미지 | 본영상에 얹을 로고 이미지 (png/jpg). 비우면 안 넣음 | 비움 |
| 마크 위치 | 좌상단 / 우상단 / 좌하단 / 우하단 | `우상단` |
| 원본 영상 보관 폴더 | 내려받은 원본 영상을 삭제하지 않고 보관할 폴더 (비우면 처리 후 삭제) | 비움 |
| 구간 후보만 분석 | 영상을 만들지 않고 하이라이트 후보만 뽑아 **수동 하이라이트 탭에 자동 입력** (Whisper를 건너뛰어 빠름) | 꺼짐 |

**버튼: 「다운로드 & 요약」 클릭** → 다운로드 → 분석 → 편집 순서로 자동 진행
완료 시 출력 폴더에 세 파일 생성:
- `제목_summary.mp4` — 요약 영상
- `제목_summary.srt` — 자막 파일
- `제목_chapters.txt` — 유튜브 챕터 (내용을 설명란에 붙여넣으면 구간 이동 챕터 생성)

> medium / large 모델은 첫 사용 시 인터넷으로 자동 다운로드되어 `models` 폴더에 저장됩니다.
> **원본 영상 보관 폴더**를 지정하면 원본이 삭제되지 않아, 나중에 **수동 하이라이트 탭**에서 다시 편집할 수 있습니다.

### STEP 1-B — 수동 하이라이트 탭 (내 영상으로 직접 만들기)

URL 다운로드·자동 분석 대신, **이미 가지고 있는 영상 파일**과 **직접 고른 시간대**로 요약본을 만들고 싶을 때 사용합니다. (요약 탭의 대안)

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 영상 파일 | 편집할 로컬 영상(mp4 등) 선택 | — |
| 출력 폴더 | 결과 파일이 저장될 폴더 | `output` |
| 출력 이름 | 결과 파일 이름(비우면 원본 파일명 사용) | 비움 |
| 하이라이트 시간대 | 한 줄에 하나씩 `시작 - 끝` (뒤에 `| 소제목` 선택) 입력 | — |
| 화면 전환 / 효과음 | 요약 탭과 동일 | `암전` / `휙` |
| 채널 마크 이미지 / 위치 | 본영상에 얹을 로고 + 모서리 | 비움 / `우상단` |
| 자막 생성 | 완성본에서 Whisper 자막(SRT) 자동 생성 on/off | 꺼짐 |

시간대 입력 형식은 `SS` / `MM:SS` / `HH:MM:SS`를 모두 지원하고, 뒤에 `| 소제목`을 붙일 수 있습니다:
```
1:23 - 2:05 | 다운그레이드
83 - 125
00:01:23,000 --> 00:02:05,000 | 보스전
```
> 구분자는 `-` `~` `->` `-->` 모두 허용하며, `#`로 시작하는 줄은 메모로 무시됩니다.
> `|` 뒤에 소제목을 적으면 그 구간 동안 **채널 마크 아래에 소제목**으로 표시됩니다(마크가 없어도 표시). 소제목이 필요 없으면 `|`를 생략하면 됩니다.

**버튼: 「하이라이트 만들기」 클릭** → 입력한 구간만 잘라 이어붙여 `이름_highlight.mp4`(+ `이름_highlight.srt`, `이름_chapters.txt`)를 만듭니다.

> **Tip:** 영상 요약 탭에서 **「구간 후보만 분석」**을 켜고 실행하면, 자동으로 찾은 후보 구간이 이 탭에 채워집니다. 구간을 확인·수정한 뒤 만들기만 누르면 됩니다.

### STEP 1-C — 쇼츠 만들기 탭 (세로 영상)

가로 영상에서 잘 나온 장면 하나를 골라 **유튜브 Shorts / Reels / TikTok용 9:16 세로 영상**으로 만듭니다.

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 영상 파일 | 원본(또는 요약본) 로컬 영상 선택 | — |
| 쇼츠 구간 | 한 줄에 하나씩 `시작 - 끝` (여러 줄은 하드컷으로 이어붙임, 총 3분 이하 권장) | — |
| 세로 변환 방식 | 중앙 크롭(가운데 확대) / 블러 배경(원본 그대로 + 위아래 블러) | `중앙 크롭` |
| 자막 크기 | 새겨넣는 캡션 크기 | `20` |
| 자막 자동 생성 | Whisper 자막을 크고 굵은 쇼츠 캡션으로 새겨넣기 on/off | 꺼짐 |

**버튼: 「쇼츠 만들기」 클릭** → `이름_shorts.mp4` (1080x1920 세로 영상)가 만들어집니다.

### STEP 2 — 자막 파일(.srt) 수정 (선택, 강력 추천)

AI가 자동 생성한 자막에는 게임 고유명사·인명 등이 틀리게 인식되는 경우가 있습니다. 완성 영상을 만들기 전에 자막을 직접 확인·수정하면 훨씬 완성도 높은 영상이 됩니다.

1. `output` 폴더에서 `제목_summary.srt` 파일을 **메모장으로 열기**
2. 틀린 단어나 어색한 문장을 수정하고 저장 (`Ctrl+S`)
3. SRT 형식 예시:
   ```
   1
   00:00:03,200 --> 00:00:06,800
   안녕하세요, 디아블로2 공략 시작합니다.

   2
   00:00:07,100 --> 00:00:10,400
   오늘은 바알 웨이브 빠르게 클리어해볼게요.
   ```
   → 번호와 시간(타임코드)은 그대로 두고, **텍스트 줄만 수정**하세요.

---

### STEP 3 — 완성 영상 만들기 탭

요약 영상 + (수정된) 자막 + 썸네일 이미지를 합쳐 업로드용 완성 mp4를 만듭니다.

| 항목 | 설명 |
|------|------|
| 영상 파일 | STEP 1에서 만든 `_summary.mp4` 선택 |
| 자막 파일(SRT) | STEP 2에서 수정한 `.srt` 선택 (수정 안 했으면 원본 선택) |
| 썸네일 이미지 | `.jpg` / `.png` 이미지 선택 (유튜브 썸네일 등) |
| 인트로 영상 (선택) | 본편 맨 앞에 붙일 별도 영상 파일 (없으면 비움) |
| 아웃트로 영상 (선택) | 본편 맨 뒤에 붙일 별도 영상 파일 (없으면 비움) |
| 배경음악 (선택) | 영상 전체에 깔 배경음악 파일 (`.mp3`/`.m4a`/`.wav` 등, 없으면 비움) |
| 출력 파일 | 최종 완성 파일 저장 경로 지정 |
| 인트로 길이(초) | 썸네일이 영상 맨 앞에 몇 초간 표시될지 (기본 `2.5`초) |
| 자막 크기 | 화면에 새겨넣을 자막의 폰트 크기 (기본 `24`) |
| 배경음악 볼륨 | BGM 크기(0~1). 말소리 대비 비율 (기본 `0.25`) |
| 인트로 삽입 | 썸네일을 첫 장면으로 넣기 on/off |
| 표지(커버) 삽입 | mp4 파일에 앨범아트처럼 썸네일 내장 on/off |
| 자막 새겨넣기 | 자막을 영상에 직접 굽기(하드섭) on/off |

**버튼: 「완성 영상 만들기」 클릭** → 자동으로 합쳐져 완성 mp4가 저장됩니다.

## 📥 설치 방법

### A) 무설치 배포판 (추천 — 파이썬 설치 불필요)
파이썬·ffmpeg·Whisper 모델까지 통째로 담은 **무설치 zip(약 1GB)** 을 받아 압축을 풀고 `실행.bat` 을 더블클릭하면 바로 실행됩니다.

> 무설치 zip은 **[👉 GitHub Releases](../../releases)** 에서 내려받으세요.

### B) 소스로 직접 실행 (파이썬 사용자)
```bash
pip install -r requirements.txt
python 요약기_gui.py        # 또는 요약기_실행.bat 더블클릭
```

**준비물**
- Python 3.9+
- ffmpeg (PATH에 등록되어 있거나, 스크립트 폴더 옆 `ffmpeg/bin/ffmpeg.exe` 로 배치)

## 🧠 동작 원리

- **하이라이트 탐지**: 오디오를 분석해 음량(에너지)이 높은 구간을 점수화하고, 목표 길이에 맞게 상위 구간을 선택합니다. 가까운 구간(`--bridge-gap` 이내)은 한 장면으로 이어붙입니다.
- **자막 생성**: OpenAI Whisper로 음성을 인식해 SRT를 만듭니다. 게임 용어 정확도를 위해 `initial_prompt` 로 도메인 단어를 힌트로 줍니다.
- **무손실 이어붙이기**: 인트로·본편을 동일 규격의 MPEG-TS 조각으로 인코딩한 뒤 `concat` 디먹서로 합쳐, 재인코딩 없이 정확한 길이로 결합합니다.
- **배경음악 삽입**: BGM을 `-stream_loop` 로 무한 반복시켜 완성 영상 전체 길이에 맞춘 뒤, 원본 오디오와 섞어(`amix`) 영상 길이에 맞게 잘라 넣습니다.
- **콘솔 창 숨김**: Windows에서 ffmpeg/yt-dlp 하위 프로세스가 검은 창을 띄우지 않도록 `CREATE_NO_WINDOW` + `-nostdin` 을 적용했습니다.

## 📁 저장소 구성

```
├── 요약기_gui.py       # GUI (영상 요약 · 수동 하이라이트 · 쇼츠 · 완성 영상 4개 탭)
├── summarizer.py       # 방송 하이라이트 요약 엔진 (+구간 분석 전용 모드)
├── manual_highlight.py # 로컬 영상 + 수동 시간대 → 하이라이트 영상
├── shorts.py           # 구간 → 9:16 세로 쇼츠 영상
├── finalize.py         # 영상+자막+썸네일+BGM 합치기
├── 요약기_실행.bat    # Windows 실행 런처 (시스템 파이썬)
├── 사용설명서.txt     # 무설치판 사용 설명서
├── assets/           # README 이미지(배너·워크플로우)
├── requirements.txt
├── LICENSE
└── README.md
```

> 무설치 배포 번들(약 1.8GB — Python·ffmpeg·Whisper 모델 포함)과 zip은 용량이 커서 저장소에 포함하지 않으며, [GitHub Releases](../../releases) 로 별도 제공합니다.
